# Piracy
An adventure game for the Arduboy.

![Screenshots](/images/screens.png)

\
Instructions:
```
  A          - Select / Confirm
  B          - Menu / Cancel
  LEFT/RIGHT - Turn ship
  UP/DOWN    - Increase/decrease speed
```  

\
Compiled with Arduino 1.8.16\
